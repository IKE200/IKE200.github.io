PORTFOLIO

Avatar - Master of Elements

The game Avatar - Master of Elements is a VR 1 vs 1 multiplayer fighting game. It recognises the movement of the player using a Microsoft Kinect and executes different elemental attacks depending on this movement. I worked mainly on implementing tools to record ground truth data to match the players movement against, as well as on the multiplayer synchronization.

Using this game as a foundation, we also conducted a small reaserch study regarding user preference of UI in virtual reality. We compared a conventional HUD vs an intradiegetic approach. Agents and Wardens

Agents and Wardens is a 1 vs 1 assymetric multiplayer game. It was created in a team of four for a masters course at Hochschule Bonn-Rhein-Sieg. One player controlls an agent in first person and has to steal secret documents and finally escape the facility. The other player plays the warden from a top down perspective. The warden can only interact with the world by giving orders to guards and by placing traps for the agent. While the agent tries to remain hidden, the warden must catch the agent.

In this project I worked mainly on the game mechanics, implementing the top down controller for the warden, as well as the alert system, which reacts to the agent making his presence known in some way to the warden. I also took over some parts in the multiplayer synchronization.
